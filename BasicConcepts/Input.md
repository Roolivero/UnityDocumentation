# Input

Input lets the game receive and respond to player actions, such as keyboard, mouse, touch, or controller input.

Player actions are translated into gameplay behavior such as movement, shooting, or interacting with objects.

Input is usually handled through scripting.

## Input and Gameplay

Input logic is often processed every frame, for example inside `Update()`.

Common gameplay uses include:

- Moving a character.
- Rotating a spaceship.
- Shooting.
- Interacting with objects.
